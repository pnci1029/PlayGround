# 로드맵 — 피드 필터 / 인기글

> 추후 기능. 둘러보기(피드)에서 **장르별 필터 + 조회수 기반 정렬/인기글**을 보고 싶다는 요구. (작성: 베타 기간, 미구현)

## 현재 상태
- 피드 정렬: `?sort=recent|popular`(인기 = `view_count DESC`)는 **이미 지원**.
- 피드 노출: 공개 글 + 내 글(비공개 포함).
- 없는 것: **장르별 필터**, 본격 **인기/랭킹 뷰**, 기간(오늘/주간) 인기.

## 추가하고 싶은 것
1. **장르 필터** — `?genre=fantasy` 등으로 특정 장르만. (FE: 피드 상단 장르 칩/드롭다운)
2. **조회수 기반 인기글** — 인기 정렬 강화: 기간별(오늘/이번 주/전체) 인기, 최소 조회수 컷.
3. (선택) **트렌딩** — 최근 N시간 조회 급상승. trend 프로젝트 방식 참고 가능.
4. (선택) **세부 장르 필터** — 세부 장르를 DB에 저장하면(현재 미저장) 그 단위 필터도 가능.

## 구현 영향 (가벼움~중간)
- **API**: `GET /api/stories`에 `?genre=`, `?period=today|week|all` 파라미터 추가. WHERE/ORDER 확장.
- **인덱스**: `(genre, view_count DESC)`, `(is_public, view_count DESC)` 등 정렬용 인덱스 추가 권장.
- **트렌딩/기간 인기**: 조회 로그(시간별)나 별도 집계 테이블 필요 → 중간 작업. 단순 "전체 조회수 정렬"은 가벼움.
- **세부 장르 필터**: 먼저 `stories.sub_genre` 컬럼 저장 필요(현재 생성 힌트로만 사용, 미저장).

## 권장 순서
1. **장르 필터 + 기간별 인기 정렬**(가벼움) — 우선.
2. 트렌딩/세부장르 필터(데이터 적재 필요) — 다음.

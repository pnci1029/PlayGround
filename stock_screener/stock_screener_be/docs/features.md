# 기능 현황

## 완성된 기능

### 스크리너 (v0.1)
- [x] 종목 전체 목록 테이블 (US 30 + KR 30)
- [x] 컬럼별 오름차순/내림차순 정렬
- [x] 시장 필터 (US / KR / 전체)
- [x] 다중 조건 필터 (AND/OR 선택, 조건 행 추가/제거)
- [x] 조건 필드: price, change_pct, market_cap, per, pbr, roe, eps, div_yield, week52_high, week52_low, volume
- [x] 연산자: `>`, `<`, `>=`, `<=`, `=`
- [x] 수동 데이터 갱신 버튼
- [x] 마지막 갱신 시각 표시

### 투자전략 패널 (v0.2)
- [x] 코스톨라니 달걀모형 6단계 (A~F) 시각화
- [x] 전략 카드 그리드 (그룹별: 코스톨라니 / 전통 / 전설적)
- [x] 전략 적용 버튼 → 해당 조건으로 스크리닝 결과 즉시 표시
- [x] 전략 그룹 탭 필터
- [x] 전략별 색상 코딩

### DCA 계산기 (v0.2)
- [x] 종목 + 시작일 + 금액 + 주기(월/주) 입력
- [x] 총 투자금 / 최종 평가액 / 손익 / 수익률 / CAGR 계산
- [x] 기간별 투자 이력 차트 (최근 36기간)
- [x] US / KR 지원

### 전설적 투자자 전략 (v0.3)
- [x] 그린블라트 마법공식
- [x] 존 네프 저PER
- [x] 월터 슐로스 딥밸류
- [x] 피터 린치 GARP
- [x] 레이 달리오 올웨더
- [x] 드레먼 역발상
- [x] 하워드 막스 안전마진
- [x] 템플턴 글로벌가치

### 백테스트 (v0.3)
- [x] 전략 선택 + 기간 입력 + 시장 선택
- [x] 정밀모드: 과거 연간 재무제표로 PER/PBR/ROE 재계산
- [x] 라이트모드: 현재 지표 기준 (과거 데이터 없을 때 자동 전환)
- [x] 종목별 데이터 품질 표시 (precise 🟢 / lite 🟡)
- [x] 포트폴리오 수익률 vs 벤치마크(SPY/KOSPI200 ETF) 비교
- [x] 알파 계산 (포트폴리오 - 벤치마크)
- [x] 매칭 종목 상세 목록 (가격, 수익률, 펀더멘털)
- [x] 12-thread 병렬 처리 (최대 120초 타임아웃)

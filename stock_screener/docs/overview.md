# 프로젝트 개요

## 목적

StockScreen은 **개인 투자자가 전통적인 투자 이론(가치투자, 코스톨라니, 피터 린치 등)을 실제 주식 데이터에 적용해볼 수 있는 도구**다.

단순한 스크리너를 넘어, 투자 전략의 논리를 배우고 → 전략에 맞는 종목을 찾고 → 과거에 그 전략이 얼마나 작동했는지 검증하는 **학습·분석 통합 플랫폼**이 목표다.

---

## 핵심 컨셉 3가지

### 1. 전략 기반 스크리닝
필터 조건을 직접 조합하거나, 유명 투자자의 전략(그린블라트 마법공식, 피터 린치 GARP 등)을 한 번에 적용해 조건에 맞는 종목을 즉시 확인한다.

### 2. 백테스트로 검증
"이 전략으로 3년 전에 투자했으면 얼마를 벌었나?"를 과거 재무제표 데이터를 활용해 계산한다. 정밀모드(과거 PER·PBR 재계산)와 라이트모드(현재 지표 기준)를 구분해 데이터 한계를 명시한다.

### 3. 내 종목 분석
관심종목 그룹을 직접 만들어 이름을 붙이고, 해당 종목들이 어떤 전략 기준에 부합하는지 + 과거부터 지금까지 수익률이 어떻게 됐는지 한눈에 파악한다.

---

## 대상 사용자

- 주식 공부를 시작한 개인 투자자
- 특정 투자 철학(가치투자, 모멘텀 등)을 실제 데이터로 검증하고 싶은 사람
- 자신의 보유 종목 / 관심 종목을 체계적으로 관리하고 분석하고 싶은 사람

---

## 철학

- **무료 데이터만 사용**: yfinance, FinanceDataReader로 구동. 유료 API 없이 동작.
- **투명한 데이터 한계 표시**: 과거 데이터 부족 시 "라이트모드"로 명시. 숫자를 과신하지 않도록.
- **모바일 우선**: 대부분의 투자 확인은 스마트폰으로 한다. PC는 심화 분석용.
- **확장 가능한 구조**: 전략 추가, 종목 universe 확장, 새 분석 기능 추가가 쉽도록 설계.
